# Flipkart Laptop Data Analysis Project

## Overview

This project focuses on scraping laptop data from Flipkart and analyzing it.
---

## Tech Stack

* Python
* BeautifulSoup
* Pandas
* Matplotlib
* Seaborn

---

## Features

* Web scraping of laptop data
* Data cleaning
* Price vs Rating analysis
* Brand-wise insights
* Visualization

---

##  Sample Insights

* High RAM laptops tend to have higher prices
* Some brands offer better ratings at lower prices

---

##  How to Run

1. Install libraries:

```bash
pip install pandas matplotlib seaborn requests beautifulsoup4
```

2. Run the notebook
